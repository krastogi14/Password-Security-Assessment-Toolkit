# 🔐 Password Security Assessment Toolkit

## Overview

The Password Security Assessment Toolkit is a security-focused project that demonstrates password assessment techniques for WPA2 Wi-Fi networks and password-protected PDF files.

The project evaluates password strength using controlled security testing methods, including WPA2 handshake capture, password hash extraction, and offline dictionary-based password assessment. It highlights how weak passwords can compromise both wireless network security and document confidentiality.

> **Disclaimer:** This project was conducted in an authorized lab environment for educational purposes only.

---

## Objectives

- Assess the strength of WPA2 Wi-Fi passwords
- Evaluate the security of password-protected PDF files
- Demonstrate password assessment using industry-standard tools
- Promote awareness of secure password practices

## Project Workflow

### Wi-Fi Password Assessment

1. Enable monitor mode on the wireless adapter.
2. Scan nearby wireless networks.
3. Select the target network.
4. Capture the WPA2 handshake.
5. Perform an offline dictionary-based password assessment using Aircrack-ng.

### PDF Password Assessment

1. Extract the PDF password hash using pdf2john.
2. Assess the password using John the Ripper.
3. Display the recovered password.

## Key Takeaways

- Weak passwords can be vulnerable to dictionary-based attacks.
- Strong, unique passwords significantly improve security.
- Password reuse should be avoided across different systems.
- Security testing should always be performed in authorized environments.
